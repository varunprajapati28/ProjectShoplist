A simple and interactive terminal-based Shopping List Manager built in Python.
This program allows users to add, update, delete, and view shopping items using a clean menu-driven approach.




Features - 

1 - Add new items to your shopping list

2 - Update existing items

3 - Delete items

4 - View all current items

5 - Exit anytime

6 - Input validation to avoid errors




How It Works - 

When you run the program:

1. You enter how many initial items you want to add.


2. The program stores them in a list.


3. A menu appears repeatedly, allowing you to:

Add more items

Update any item

Delete items

View all items

Exit



How to Run

1. Install Python (if not already installed).


2. Save the script as shoplist.py.


3. Open your terminal or VS Code and run:

python shoplist.py
